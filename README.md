OWL Icons are the default icon set of [OWL UI](https://github.com/cepave-f2e/vue-owl-ui).

## Install

```sh
$ yarn add @cepave/owl-icons
```

## Deploy

- Add new icon svg in `/icons`.
- Update new version in `package.json`.
- git commit and push to `master`.

That's all
